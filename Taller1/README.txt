Archivo p1.c crea un autómata celular con regla 30 de n filas. La forma de compilar el archivo es

	p1.c -o codigoTaller1 n
	
Donde n es el número de filas

Archivo p2.c crea un autómata celular con regla r de n filas. La forma de compilar el archivo es

	p1.c -o codigoTaller1 r n
	
Donde r el numero de la regla entre 0 y 255, y n es el número de filas.
